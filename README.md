# 养老服务管理系统 (Yanglao Service Management System)

本项目是一个基于前后端分离架构的综合性养老服务管理平台，旨在为老年人、家属及管理员提供便捷、高效的养老服务体验。

## 1. 系统架构

*   **前端**：采用 Vue 3 + Vite + Element Plus 构建，提供响应式、现代化的用户界面。
*   **后端**：采用 Java Spring Boot 微服务架构，结合 Spring Cloud Alibaba (Nacos) 进行服务治理。
    *   `administrator-server` (5001): 管理员服务，负责系统管理、用户管理及数据统计。
    *   `login-server` (5000): 认证服务，负责统一的注册与登录认证。
    *   `user-server` (5002): 用户服务，负责用户个人信息及家属关联管理。
    *   `business-server` (5003): 业务服务，负责生活服务项目管理及订单处理。
    *   `common-server` (5004): 公共服务，负责公告通知管理。
    *   `activity-server` (5005): 活动服务，负责社区活动的发布与报名管理。
    *   `message-server` (5006): 消息服务，负责留言板及互动交流。
    *   `health-server` (5007): 健康服务，负责健康档案及体征数据管理。

## 2. 角色与功能

系统支持三种角色：**用户（老人）**、**家属** 和 **管理员**。

### 2.1 用户（老人）
*   **个人中心**：查看和修改个人基本信息，修改登录密码。
*   **服务预约**：浏览各类生活服务（如家政、助餐），提交预约订单，并对已完成的服务进行评价。
*   **活动参与**：浏览社区活动列表，在线报名参与，查看我的报名记录及取消报名。
*   **健康管理**：查看个人健康档案，录入每日体征数据（血压、心率等），查看历史健康数据趋势。
*   **互动交流**：查看社区公告，在留言板发布留言或建议。

### 2.2 家属
*   **长辈关怀**：绑定并查看关联老人的基本信息。
*   **健康监护**：查看老人的健康档案及历史健康数据，及时掌握老人身体状况。
*   **互动交流**：查看公告，发布留言反馈。

### 2.3 管理员
*   **工作台**：查看系统核心数据统计（用户数、订单数等），使用快捷操作入口。
*   **用户管理**：对老人、家属及管理员账号进行增删改查管理。
*   **服务管理**：发布、编辑、上下架服务项目；审核、接单、完成或取消服务订单。
*   **活动管理**：发布新活动，管理活动状态，查看活动报名情况。
*   **健康管理**：建立和维护老人的电子健康档案，监控全平台健康数据。
*   **公告管理**：发布、编辑和删除系统公告。
*   **留言管理**：查看并回复用户留言，删除违规内容。

## 3. 核心业务模块

### 3.1 认证模块
*   支持多角色登录与注册。
*   注册时支持家属与老人的关联校验。

### 3.2 生活服务模块
*   **服务展示**：图文并茂展示服务详情及价格。
*   **订单流程**：预约 -> 待确认 -> 已确认 -> 已完成/已取消。
*   **评价体系**：服务完成后可进行评分和评论。

### 3.3 活动模块
*   **状态管理**：自动区分报名中、进行中、已结束的活动。
*   **报名限制**：支持人数限制及重复报名校验。

### 3.4 健康模块
*   **档案管理**：记录身高、体重、血型等基础信息。
*   **数据追踪**：记录并展示血压、血糖、体温等多维度的时序数据。

## 4. 技术栈

*   **前端**：Vue 3, Vite, Pinia, Vue Router, Element Plus, Axios, ECharts
*   **后端**：Spring Boot 3, Spring Cloud Alibaba, MyBatis-Plus, MySQL, Druid, OpenFeign
*   **工具**：Maven, Git, Nacos

## 5. 快速开始

1.  **环境准备**：JDK 17+, Node.js 16+, MySQL 8.0+, Nacos 2.x。
2.  **数据库初始化**：执行 `sql/` 目录下所有 SQL 脚本。
3.  **后端启动**：
    *   启动 Nacos。
    *   依次启动各微服务模块 (`AdministratorApplication`, `LoginApplication` 等)。
4.  **前端启动**：
    *   进入 `yanglao-project` 目录。
    *   运行 `npm install` 安装依赖。
    *   运行 `npm run dev` 启动开发服务器。
5.  **访问**：浏览器打开 `http://localhost:3000`。
